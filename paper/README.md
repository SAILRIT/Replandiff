# Repair, Don't Replan — NeurIPS-workshop paper source

Build: `make` (or `latexmk -pdf main.tex`). Compiles with TeX Live, no network needed.

```
main.tex             paper source (body + appendix)
neuripsstyle.sty     NeurIPS-compatible layout  <-- see note 1
refs.bib             bibliography               <-- see note 2
tab_main.tex         Table 1, generated from results.json by gen_paper_tables.py
tab_family.tex       Appendix Table 2, likewise generated
figures/*.pdf        vector figures, generated by plots.py at 5.5in text width
Makefile
```

### 1. The style file is a reimplementation, not the official one
The official NeurIPS `.sty` could not be downloaded in the build sandbox, so
`neuripsstyle.sty` reproduces the layout (5.5in x 9in text block, 10pt Times, 17pt title,
NeurIPS float spacing and heading sizes). **Before submitting**, download your workshop's
official style file and swap one line:

```latex
\usepackage[preprint]{neuripsstyle}   ->   \usepackage{neurips_2026}   % or the workshop's
```

Nothing in the body needs to change. `\noticeline{...}` sets the first-page footnote;
the official styles set that themselves, so delete that line after swapping.

### 2. Bibliography status
All entries now carry verified author lists and titles (checked against arXiv abstract
pages July 17, 2026); no `Anonymous` entries remain. Two entries retain `and others`
(Agent-Diff, DiffuCoder) pending full lists. Re-verify against arXiv/DBLP at submission
time as a matter of hygiene.

### 3. Page limit
The main body runs ~5 pages; references and appendix follow. If your workshop caps the body
at 4 pages (excluding references/appendix), the intended cuts, in order:
1. Move §5 Related work to the appendix (~0.35 pp).
2. Move Figure 2 (Pareto) to the appendix — Table 1 already carries those numbers (~0.4 pp).
3. Fold the "Regeneration exposure" and "Typed holes" paragraphs into one sentence each,
   pointing at the appendix figures (~0.25 pp).
Do not cut the AR-patch paragraph or the Scope/Limitations text: they are what make the
claims defensible.

### 4. Author block
`\author{...}` in `main.tex` is a placeholder. Fill it in, or add your workshop's
anonymisation option when swapping the style file.

### 5. Provenance of every number
All results come from `replandiff_code.zip` (separate deliverable): synthetic ToolFlow-v2
environment, ~250k-parameter models trained from scratch, 5 seeds (3 for the RL condition),
400 episodes/cell. `tab_main.tex` and `tab_family.tex` are emitted directly from
`results/results.json` by `gen_paper_tables.py`, so the table cannot drift from the data.
Regenerate with: `python3 gen_paper_tables.py && python3 plots.py`.

The paper states its scope in §1 ("Scope") and §6: this is a controlled mechanism study on a
synthetic environment, **not** an evaluation of 7-8B dLLMs on Agent-Diff/BFCL. Keep that
framing — it is accurate, and it is what makes the rest of the paper credible.

## Second strengthening round (this revision)
- **Supervision-dose intervention (new §4 paragraph + Appendix D + Fig. 8).** The paper's
  central conjecture — priority rules must be in the supervision — is now a tested result:
  a self-contained cov=1.0 grid (doses k∈{0,8,64}; 5/5/3 seeds) shows the composed
  conditional p(escalate|SUSPENDED+INV_OPEN) goes 0.408±0.341 (1/5 dominance-consistent)
  → 0.997±0.002 (5/5) with k=8 (Fisher exact one-sided p=0.024); famA doubles 0.59→1.00;
  k=64 saturated. The k=0 arm carries 10x the base models' single-perturbation repair
  supervision, making it a stronger control than "more of the same data".
- **Mechanistic probe (§4).** The same composed-conditional probe on the five base models
  (0.85/0.05/0.96/0.62/0.16) predicts each seed's end-to-end famA outcome by sign — and
  explains the oracle's win: dominance-aware localization never re-queries the unreliable
  conditional.
- **Masker operating curve (Fig. 9, Appendix B).** tau_inv swept over [0.10,0.65] x 5 seeds
  x n∈{1,2}: success flat everywhere; no threshold rescues composition.
- **Compute-only control (Appendix D).** Warm-starting the two dominance-failing base
  seeds for a second full training budget leaves famA at 0.000 and hardens the composed
  conditional against the rule (probe 0.05->0.000, 0.16->0.001) while famC/n1/n0 stay at
  1.000: compute entrenches the coin flip; demonstrations decide it.
- **Disclosed anomalies (Appendix D).** An initial dose attempt at cov=0.1 hit a
  supervision-scarcity training lottery (one repair-degenerate draw); 3/5 cov=1.0 k=0 draws
  corrupted an unrelated benign conditional (persists at tau=0.15, so not a threshold
  artifact); all five k=8 draws are clean — regularization interpretation flagged as
  speculative. Dose-grid numbers are within-grid only (1500 steps, cov=1.0 ≠ base config).

## Scale study (this revision)
- **Model-scale x dose factorial (Appendix D, Fig. 10).** Grid repeated at 46,144 params
  (d=48, 2L; 0.19x base) and 526,384 params (d=144, 3L; 2.1x), plus a 4x-data arm (16k
  instances) at base width. Dose-0 draws are dominance-consistent 3/13 across all
  configurations; every healthy dose-8 draw is 10/10 (exploratory pooled Fisher p=2.5e-4).
  The threshold-crossing rule (end-to-end famA success iff the composed conditional clears
  tau_inv) holds on every healthy cell; the sec-4 sentence was upgraded from "sign" to the
  threshold formulation.
- **Disclosures:** one 526k dose-8 draw excluded for a documented mid-training loss spike
  (replaced by a clean seed, shown as x in Fig. 10); one 46k dose-8 draw shows the famC
  conditional corruption, retiring the "demonstrations regularize" speculation; 4x-data
  draws are corruption-free while the composed rule stays a lottery. The sweep licenses a
  46k-526k scale-trend claim and nothing about 7-8B models (stated in Limitations).
